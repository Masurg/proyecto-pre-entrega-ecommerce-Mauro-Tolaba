# Mat Shop - Proyecto Ecommerce

## Descripción
Esto es un proyecto pensando en un Ecommerce.
El Ecommerce se llama "MatShop"
El Proyecto es una "Pre-Entrega" y fue estructurando con las actividades de las clases (1 a 8).

El Proyecto cuenta con multimedia creados por mi. (videos y imagenes) de productos en este caso libros por el momento.

Cuenta con paginas como: categorias.html y contacto.html

Se trabajo con  fuentes Outfit y Nunito sacado de Fonts.Google, se puso sans-serif como fuente secundaria en caso de sin conexion.

Para los Iconos se trabajo con Font Awesome Free 6.5.0

Colores principales del Ecommerce: "#8c52ff y #5ce1e6"


## Tecnologías utilizadas
- HTML
- CSS
- Google Fonts (Outfit y Nunito)
- Font Awesome 6.5.0
- Formspree (formulario de contacto)
- GitHub Pages (hosting)

## Estado del proyecto
Este Proyecto. es una fase de Pre-Entrega.
